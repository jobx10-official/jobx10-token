# Token Utility

The **JX10 Token** enables multiple functions within the JobX10 ecosystem.

## Payments
Users can pay for:
- Coaching programs  
- Digital learning modules  
- JobX10 premium services  

## Discounts
Holders receive:
- Reduced coaching fees  
- Partner discounts (future integrations)

## Rewards
Community members can earn JX10 through:
- Participation  
- Engagement  
- Referral activities  

## Access
The token unlocks access to:
- Premium content  
- Early product features  
- Future mobile app utilities  

More use cases will be introduced as the ecosystem expands.
