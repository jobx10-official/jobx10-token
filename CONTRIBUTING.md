# Contributing to JobX10 (JX10)

Thank you for your interest in contributing to the JobX10 Token project!  
We appreciate community support and welcome improvements that enhance the safety, clarity and usability of the ecosystem.

This document explains how to contribute and how to collaborate with the project team.

---

## üß© Types of Contributions

We welcome contributions in the following areas:

### 1. Documentation
- Improving README or docs  
- Fixing spelling or clarity  
- Adding missing project explanations  
- Expanding developer notes  

### 2. Technical Improvements
- Suggestions for contract optimization  
- Adding tests or analysis tools  
- Improving repository structure  

*(Note: Smart contract code is non-upgradable. No direct contract changes will be deployed.)*

### 3. Bug Reports
- Issues in documentation  
- Metadata or repository errors  
- Whitepaper corrections  
- Website content problems  

### 4. Community Support
- Suggestions for ecosystem development  
- Partner integrations  
- Feedback for future features
